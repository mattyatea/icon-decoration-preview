# Misskey Icon Decoration Preview
Misskey v2023.11.0で実装予定のアイコンデコレーションのプレビューを見れるやつです。

Node v20で動作確認済です

# Installation
適当なところにcloneして

`pnpm install`

して

`pnpm build`

して

` node .output/server/index.mjs `

すれば起動します
